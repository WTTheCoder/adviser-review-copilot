# Fictional PDF demo source

Alex may have moved from East Perth to Joondalup and the new address still needs confirmation.

Alex is considering changing to a High Growth risk profile.

## Generate the text-based PDF

Open this Markdown file in an editor or browser that can render Markdown, then use **Print > Save as PDF**. Confirm that the resulting PDF text can be selected before uploading it. Save it as `alex-taylor-update.pdf`.

Do not scan or rasterize the page: the prototype supports embedded text and does not include OCR.
